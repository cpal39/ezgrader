# [EZgrader](https://cpal39.github.io/ezgrader/)

A simple tool to see what grade should be given on a test where all questions are worth the same amount of points.

## Why this exists

My mother is a teacher and wanted an easy grader she could quickly access. Love you Mom!

## Usage

Enter the number of questions that are on the test, and the grading breakdown will be displayed automatically as the input changes.
